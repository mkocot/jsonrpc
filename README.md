# jsonrpc
Jsonrpc is library for handling JSON-RPC 2.0 requests.
For documentation see [https://mkocot.github.io/jsonrpc/jsonrpc/index.html](https://mkocot.github.io/jsonrpc/jsonrpc/index.html).
